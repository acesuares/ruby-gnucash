# Gnucash

Ruby library for extracting data from GnuCash data files

## Installation

Add this line to your application's Gemfile:

    gem 'gnucash'

And then execute:

    $ bundle

Or install it yourself as:

    $ gem install gnucash

## Usage

    book = Gnucash.open("MyBook.gnucash")
